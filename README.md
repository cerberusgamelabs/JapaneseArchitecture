# Japanese Architecture
Japanese Architecture is a content-driven mod for Vintage Story, created by [LilianaXolkiyr](https://mods.vintagestory.at/show/user/B637160E994DD71C94F2). This mod introduces a variety of blocks and assets that allow players to construct authentic Japanese-style structures such as traditional homes, shrines, and temples.
